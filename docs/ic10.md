# Stationeers IC10 LSP for Neovim

## Example Code (Solar Tracking)
```ic10
# Simple Solar Tracking
alias SolarSensor d0
alias SolarPanel d1

alias horz r0
alias vert r1

Main:
  l horz SolarSensor Horizontal
  l vert SolarSensor Vertical

  s SolarPanel Horizontal horz
  s SolarPanel Verical vert

  yield
  j Main
```
